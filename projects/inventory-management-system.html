01 / OVERVIEW

This project involved the design and development of a full-stack inventory management system for managing products, inventory levels, and stock transactions.

The system combined a React frontend with a Django backend and SQLite database to provide a structured platform for managing inventory-related operations.

02 / MY ROLE

I served as the Project Manager for this project, coordinating the development process and contributing to the overall planning and implementation of the system.

  • Participated in project planning and task coordination.
  • Contributed to system design and development.
  • Worked on backend and database integration.
  • Supported testing and debugging activities.
  • Helped coordinate project documentation and deliverables.

03 / KEY FEATURES

  • Product and inventory management.
  • Stock transaction tracking.
  • CRUD operations for inventory data.
  • Input validation for user-submitted data.
  • User authentication.
  • Role-based access control.
  • Database integration using SQLite.
