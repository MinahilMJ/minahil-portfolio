Make
Squares

A strategic board game developed in Python and Pygame, featuring an AI opponent that uses the Minimax algorithm with Alpha-Beta Pruning to evaluate moves and compete against the player.

Python Pygame Artificial Intelligence Minimax Alpha-Beta Pruning

01 / OVERVIEW

Make Squares is a strategic board game based on the classic dots-and-boxes concept. Players take turns drawing horizontal or vertical lines on a grid, with the objective of completing squares and earning points.

The project combined game development with artificial intelligence by implementing an AI opponent capable of evaluating possible moves and selecting strategic actions.

02 / AI IMPLEMENTATION

The AI opponent used the Minimax algorithm with Alpha-Beta Pruning to evaluate possible game states and select moves.

  • Evaluated possible future moves.
  • Used Minimax for decision-making.
  • Applied Alpha-Beta Pruning to reduce unnecessary search.
  • Selected moves based on evaluated game states.
  • Supported strategic gameplay against a human player.

03 / GAME SYSTEM

  • Players could draw horizontal and vertical lines on a grid.
  • Completed boxes were detected automatically.
  • Players earned points by completing squares.
  • The game tracked the board state throughout the match.
  • Board dimensions could be defined interactively.
  • The game continued until all possible moves were completed.
