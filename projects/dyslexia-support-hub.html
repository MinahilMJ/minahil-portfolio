BS FINAL YEAR PROJECT / 01

Dyslexia Support Hub

An AI-assisted educational platform designed to support children with dyslexia through personalized learning, adaptive educational content, and machine learning-based recommendations.

React Django Python PostgreSQL Machine Learning REST APIs

01 / OVERVIEW

Dyslexia Support Hub was developed as an AI-assisted educational platform to support children with dyslexia through personalized and adaptive learning experiences.

The platform was designed to adjust educational content based on individual student performance and learning progress, creating a more personalized learning experience.

02 / PLATFORM FEATURES

  • Adaptive learning modules.
  • Personalized educational experiences.
  • Machine learning-based performance analysis.
  • Personalized learning recommendations.
  • Interactive learning activities.
  • Student progress tracking.
  • Performance dashboards.
  • Secure user authentication.

03 / SYSTEM DEVELOPMENT

The platform was developed as a full-stack web application using React for the frontend and Django for the backend.

REST APIs were implemented to facilitate communication between the frontend and backend systems, while the platform used a database to manage user information, learning progress, and related application data.

04 / MACHINE LEARNING

Machine learning techniques were integrated to analyze student performance and support the generation of personalized learning recommendations.

The system used student performance and learning progress as inputs to support adaptive educational experiences.
